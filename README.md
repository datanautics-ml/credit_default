# credit_default

Lightweight Python project for exploring, training, and evaluating credit default prediction models.

## Project overview
- Purpose: preprocess credit data, train classification models to predict borrower default, and evaluate model performance.
- Contents: data ingestion, feature engineering, model training, evaluation, and utilities for experiment reproduction.

## Repository layout
- data/                     — (gitignored) expected raw and processed datasets
- notebooks/                — EDA and prototyping notebooks
- src/                      — core Python modules
    - training.py
    
- models/                   — saved model artifacts
- requirements.txt
- README.md

## Requirements
- Python 3.10+
- Install dependencies:
    ```
    pip install -r requirements.txt
    ```
